# 实施计划: 万能WiFi钥匙 (网页版)

## 目标描述
构建一个功能强大的“万能WiFi钥匙”网页应用。目标不仅仅是连接，而是集成**核心WiFi破解与密钥获取功能**。界面风格保持高端“赛博朋克”风格，打造黑客级的WiFi工具体验。

**核心重点:**
- **核心功能:** WiFi密码破解 (字典攻击/PIN破解)、云端密钥共享、安全漏洞检测。
- **视觉:** 黑客终端风格结合毛玻璃美学，强调数据流与解密过程的视觉反馈。
- **平台:** Web (HTML/CSS/JS)，通过未来适配层 (如 Native/WASM) 实现底层渗透能力。

## 用户审查必需项
> [!WARNING]
> **功能合规提示**: 本项目包含渗透测试与密码破解功能，请确保仅用于合法授权的安全测试或自己的网络调试。
> **技术实现**: 网页端将优先实现**云端密钥库查询** (万能钥匙模式) 和 **字典/PIN 算法生成的逻辑层**。底层网卡调用将在后续 App 封装阶段通过 Native 接口接入。

## 拟定设计与功能

### 1. 视觉识别
- **主题**: “黑客帝国”绿 (#00ffa3) 与 “赛博格”蓝 (#00f3ff) 混搭，深色黑洞背景。
- **风格**: 终端命令行 (Terminal) 元素，动态解码动画，全息投影效果。

### 2. 功能模块
1.  **首页 (渗透中心)**:
    -   中央“一键破译”按钮 (AUTO CRACK)。
    -   实时周围信号漏洞扫描雷达。
    -   破解日志滚动显示 (Console Log)。
2.  **WiFi 列表 (目标选择)**:
    -   显示加密类型 (WPA2/WPA3) 及漏洞评级 (高危/可破)。
    -   针对特定热点的攻击选项: 字典爆破 / WPS PIN / 云端查询。
3.  **破解工具箱 (Attack Tools)**:
    -   **字典管理**: 导入/编辑密码字典。
    -   **PIN 算号器**: 根据 MAC 地址计算默认 PIN。
    -   **抓包分析**: 模拟握手包抓取过程展示。
4.  **我的 (黑客档案)**:
    -   已破解 WiFi 历史记录 (密码本)。
    -   渗透测试报告生成。

## 技术栈
- **核心**: HTML5, 原生 JavaScript (ES6+)。
- **样式**: 原生 CSS3 (CSS 变量, Flexbox/Grid, Backdrop-filter)。
- **图标**: Phosphor Icons 或 FontAwesome (CDN引入)。
- **构建**: Vite (可选，用于热重载) 或 简易 HTTP服务器。

- 本阶段不包含自动化测试 (专注于UI原型)。
