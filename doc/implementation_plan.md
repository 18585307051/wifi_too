# 实施计划: 万能WiFi钥匙 (网页版)

## 目标描述
构建一个高端的“万能WiFi钥匙”风格的网页应用。目标是创建一个视觉效果惊艳、适配移动端的网页界面，模拟WiFi万能钥匙的功能，为未来转型为原生移动应用做准备。

**核心重点:**
- **视觉:** 高端的“毛玻璃（Glassmorphism）”美学，生动的色彩，流畅的动画。
- **平台:** Web (HTML/CSS/JS), 完全响应式 (移动端优先)。

## 用户审查必需项
> [!IMPORTANT]
> **网页端限制**:以此特别说明，由于浏览器沙盒的安全限制，标准的网页端**无法实现**真实的WiFi扫描和密码破解功能。 所有的“功能”（如扫描、连接、安全检测）都将是**模拟/演示**效果，或者是基于用户手动输入/共享数据库的逻辑。本应用将着重于展现极致的*UI/UX*交互体验和数据共享平台概念。

## 拟定设计与功能

### 1. 视觉识别
- **主题**: 暗黑模式基底，搭配“科技蓝” (#00f0ff) 和 “霓虹紫” (#7000ff) 点缀。
- **风格**: 毛玻璃效果 (磨砂背景), 发光边框, 丝滑的微交互动画。
- **排版**: 现代无衬线字体 (Inter 或 Roboto)。

### 2. 功能模块
1.  **首页 (一键连接)**:
    -   巨大的呼吸灯效果“一键连接”按钮。
    -   当前连接状态 (信号强度, IP地址, 安全评分)。
    -   旋转的雷达背景动画。
2.  **WiFi 列表 (发现)**:
    -   可用网络列表 (演示数据)。
    -   “免费”, “安全”, “极速” 等标签指示器。
    -   密码共享/输入界面。
3.  **工具箱 (实用工具)**:
    -   **网络测速**: 带有仪表盘动画的 Ping/下载/上传 测试。
    -   **安全检测**: 扫描ARP攻击、DNS劫持的模拟动画。
    -   **信号地图**: 热力图风格的可视化图形。
4.  **我的 (个人中心)**:
    -   节省流量/数据统计。
    -   设置 (主题切换, 通知)。

## 技术栈
- **核心**: HTML5, 原生 JavaScript (ES6+)。
- **样式**: 原生 CSS3 (CSS 变量, Flexbox/Grid, Backdrop-filter)。
- **图标**: Phosphor Icons 或 FontAwesome (CDN引入)。
- **构建**: Vite (可选，用于热重载) 或 简易 HTTP服务器。

## 目录结构
```
/ (根目录)
├── index.html          # 入口文件 (单页应用壳)
├── /css
│   ├── main.css        # 重置与排版
│   ├── theme.css       # 变量 (颜色, 玻璃特效)
│   └── components.css  # 按钮/卡片样式
├── /js
│   ├── app.js          # 主逻辑与路由
│   ├── state.js        # 数据状态 (WiFi列表, 用户状态)
│   └── /views          # 视图控制器 (首页, 列表, 工具)
└── /assets             # 图片与图标
```

## 验证计划
### 自动化测试
- 本阶段不包含自动化测试 (专注于UI原型)。

### 手动验证
1.  **视觉检查**: 验证 Chrome/Edge 浏览器上的毛玻璃模糊效果。
2.  **响应式测试**: 通过开发者工具测试 iPhone SE/12/14 Pro 等尺寸的适配情况。
3.  **交互测试**: 确保按钮点击、标签切换和动画播放流畅无卡顿。
